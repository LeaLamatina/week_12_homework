Project Explanation

A minimum of 500 city GPS coordinates were generated and run through API calls to retrieve weather data. Calls were completed on August 9th, 2018. Information gatered included the following:

    Cloud density
    Country
    Humidity
    Latitude and Longitude
    City Name
    Temperature
    Wind Speed

The purpose of the project was to plot Temperature, Wind Speed, and Cloudiness against Latitude to determine whether there exisited a correlation.

All visualizations were created using Matplotlib.

A complete list of the raw data is available on the under the Data tab.
